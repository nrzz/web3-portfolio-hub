## Design Features

### Color Scheme
- **Primary Blue**: #2563eb (Blue-600)
- **Success Green**: #16a34a (Green-600)
- **Warning Orange**: #ea580c (Orange-600)
- **Error Red**: #dc2626 (Red-600)
- **Neutral Grays**: #f9fafb to #111827

### Typography
- **Headings**: Inter, Bold (700)
- **Body**: Inter, Regular (400)
- **Monospace**: For addresses and numbers

### Interactive Elements
- **Hover Effects**: Subtle color transitions
- **Focus States**: Blue outline for accessibility
- **Loading States**: Spinning indicators
- **Animations**: Framer Motion for page transitions

### Responsive Design
- **Desktop**: Full sidebar navigation
- **Tablet**: Collapsible sidebar
- **Mobile**: Bottom navigation bar

### Accessibility
- **High Contrast**: Dark mode support
- **Keyboard Navigation**: Full keyboard support
- **Screen Reader**: Proper ARIA labels
- **Color Blind**: Not relying solely on color

This design provides a modern, clean interface for managing Web3 portfolios with real-time data visualization and comprehensive analytics.
